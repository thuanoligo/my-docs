# VM

Oligo supports VM-based deployments through two modes: **Managed** and **Non-Managed**. Choose the deployment model that best fits your infrastructure management approach.

{% tabs %}
{% tab title="Managed" %}
In a Managed deployment, the Oligo platform handles sensor installation, updates, and configuration remotely. This is the **recommended** approach for most VM environments.

[Get started with Managed Deployment](managed-deployment.md)
{% endtab %}

{% tab title="Non-Managed" %}
In a Non-Managed deployment, you install and manage the sensor manually on each VM. Use this mode for air-gapped or restricted environments.

[Get started with Non-Managed Deployment](non-managed-deployment.md)
{% endtab %}
{% endtabs %}
